Догађаји

Оно што чини ГДевелоп уникатним и лаким за коришћењем то су догађаји. Догађаји су моћан начин да се изрази логика ваше игре, без потребе учења програмског језика.

Када је спејс (размак) притиснут, анимација карактера и тон су пуштени. Ако бомба додирне карактера, обоје су уништени.

Create HTML5, mobile and native games

GDevelop's flexible architecture allows you to choose if you want to create an HTML5 game that can be played on the web and exported to iOS and Android, or a native game for your operating system. You can also activate both platforms and export your game everywhere!

ГДевелоп је доступан заУбунту 64битнисамо!

Отвори терминал притиском Ctrl + Alt + T.

Онда одкуцај (и притисни ентер након сваке линије):

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop
